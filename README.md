# Task Management Dashboard

This project features a secure login system and a data-driven dashboard integrated with a REST API.

## Live Demo
**[Live Link](https://ocarnal.netlify.app/login)**

---

## Features
* **Secure Authentication**: Implemented JWT-based login with persistent private routes.
* **Dynamic Dashboard**: Fetched and displayed real-time data using Axios from the provided REST API.
* **Adaptive Grid Layout**: A custom 12-column responsive grid system optimized for all screen sizes (Mobile, Tablet, Desktop).

---

## Tech Stack
* **Framework**: React (Vite)
* **Styling**: Tailwind CSS
* **Icons**: Lucide React
* **API Client**: Axios
* **Routing**: React Router DOM
